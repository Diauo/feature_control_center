## 这是什么？
用于管理杂乱功能的web管理面板，包含功能管理，定时任务，日志管理，运行结果报表等

## 画大饼
- 功能管理
  - 功能分类
  - 功能数据源管理
  - 标签查询
  - 执行功能
    - 普通执行
    - 执行队列
    - 批量提交到线程池执行
  - 添加功能
    - 功能热重载
  - 基于多配置文件的功能引用
- 定时任务
  - 基本定时任务管理
  - 基于线程池和队列的可配置定时任务
- 日志管理
  - 查每个功能的运行日志（考虑引入MongoDB
- 运行结果报表
  - 每个任务结束后的报表，包括成功的关键ID集合，和失败的ID集合
  - 基于运行结果，快速执行重试
